Project (Music management system):

This project allows users to manage songs, create playlists, and handle user authentication . It performs Various operations such as adding songs, creating playlists and more.

How to run project:

1.)	Download the project:
->Place all the files (.c, .h , Make file) in the same directory. 
2.) compile the project:
->Navigate to the project folder in the terminal.
3.) create playlist:
->create a playlist and add songs to it.
4.) view playlist:
-> view all the playlists owned by the logged in user.
5.) Remove song:
-> remove a song from the library by its unique id.
6.) Exit:
-> save all the data and quit the program. 


Project structure:
->main.h                            #contains void of calling main menu function
->Makefile      		            #build instructions for the project 
->main.c			                #main entry point for the program
->songs_ library.c                  #logic for song management 
->playlists.c   	                #logic for playlist management 
->login.c  	 	                    #logic for user authentication
->utility.c   	 	                #utility functions like clearing the screen
->songs_ library.h                  #header file for songs library
->playlists.h   	                #header file for playlists
->login.h		                    #header file for user authentication
->utility.h    		                #header file for utilities
->users.txt     		            #stores user data
->playlists.txt     	            #stores playlists and song data
